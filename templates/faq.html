What does everything stand for?

Time Acronyms:

A: 8:00 - 9:20 AM
B: 9:40 - 11:00 AM
C: 11:20 AM - 12:40 PM
D: 1:00 - 2:20 PM
E: 2:40 - 4:00 PM
F: 4:20 - 5:40 PM
G: 6:00 - 7:20 PM

Day Acronyms:

MW: Monday and Wednesday
TR: Tuesday and Thursday

How are the CSVs formatted?

The two CSVs that must be uploaded are the Department Class Data and room table CSVs. These have a specific form to them that must be followed.

How is the Department Class Data CSV formatted?

The Department Class Data CSV has 6 fields. Course Selection which houses the course id that includes the department, the class number, and section (ex. CS100-01). Faculty Assignment which contains the teacher who will be teaching the class. Room Preference which contains the room that you wish the class to be taught. Time Block Preference which contains The time block you wish the class to be held at. These blocks range from A to G. You can have any combination of them, but you must have them all without spaces. Day Preference which contains the day you wish the class to be taught. This can either be MW or TR or a combination of both. Seats Open which contains the amount of seats for the given classroom.

How is the Avaliable Room CSV formatted?

The Avaliable Room CSV have only 2 fields. Building which is the building where the rooms are located and Room Number which holds the room number of a given classroom. In order for a room to be included in the creation of the schedule, the room must be included in this CSV.
