CSV Formatting

The two CSVs that must be uploaded are the Department Class Data and room table CSVs. These have a specific form to them that must be followed.

Department Class Data CSV

The Department Class Data CSV has 6 Sections. Course Selection which houses the course id that includes the department, the class number, and section (ex. CS100-01). Faculty Assignment which contains the teacher who will be teaching the class. Room Preference which contains the room that you wish the class to be taught. Time Block Preference which contains The time block you wish the class to be held at. These blocks range from A to G. You can have any combination of them, but you must have them all without spaces. Day Preference which contains the day you wish the class to be taught. This can either be MW or TR or a combination of both. Seats Open which contains the amount of seats for the given classroom.
