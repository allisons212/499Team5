What does everything stand for?

Time Acronyms:

A: 8:00 - 9:20 AM
B: 9:40 - 11:00 AM
C: 11:20 AM - 12:40 PM
D: 1:00 - 2:20 PM
E: 2:40 - 4:00 PM
F: 4:20 - 5:40 PM
G: 6:00 - 7:20 PM

Day Acronyms:

MW: Monday and Wednesday
TR: Tuesday and Thursday

How do you upload a CSV?

CSV Upload

When uploading CSVs on the Upload CSV page you must choose two files: AvailableRoomsCSV and DepartmentClassData CSV If you do not choose two files and try to Submit the CSVs then it will give you an error message. Once you give two files and they are in the correct CSV format, then it will notify you that it has successfully upload the files. After successfully uploading both files, go to Create Schedule and generate the newly updated schedule.

Manual Upload

The manual upload function is specifically for adding courses that you forgot after you have used the CSV function. There will be six fields that are required to fill out. These fields are Department, Class, Faculty Member, Room, Day, and Time. The Class must follow this format: [3-digit integer]-[2-digit integer]' EX. 102-01 where 01 indicates the section number. With the manual entry, you will also be allowed to rewrite already existing classes. If this is occuring a warning will pop up. The Faculty Member must follow this format: 40 characters or less using only letters, periods, apostrophes, and spaces. Once all fields are filled out, hit the Submit Manual Entry to add class to database. After submitting, go to Create Schedule and generate the newly updated schedule.

How are the CSVs formatted?

The two CSVs that must be uploaded are the Department Class Data and room table CSVs. These have a specific form to them that must be followed.

How is the Department Class Data CSV formatted?

The Department Class Data CSV has 6 fields. Course Selection which houses the course id that includes the department, the class number, and section (ex. CS100-01). Faculty Assignment which contains the teacher who will be teaching the class. Room Preference which contains the room that you wish the class to be taught. Time Block Preference which contains The time block you wish the class to be held at. These blocks range from A to G. You can have any combination of them, but you must have them all without spaces. Day Preference which contains the day you wish the class to be taught. This can either be MW or TR or a combination of both. Seats Open which contains the amount of seats for the given classroom.

How is the Avaliable Room CSV formatted?

The Avaliable Room CSV have only 2 fields. Building which is the building where the rooms are located and Room Number which holds the room number of a given classroom. In order for a room to be included in the creation of the schedule, the room must be included in this CSV.

How do you generate a schedule?

After the correct CSVs have been uploaded, Click the generate button on the create schedule page. This will generate a schedule from the CSVs. The schedule table will then be filled out with the created schedule.

How do you resolve conflicts?

Once a schedule has been generated, any conflicts that are created must be resolved. To solve conflicts, click on the conflict icon to see the conflict resolution window.

Conflict Resolution page:

Once in this window, you will see all the classes with conflicts. These classes need to be assigned a classroom and day and time. To start resolving the conflict click on the Classroom Selection dropdown for a given class and select a classroom. Once a classroom is selected, you can see all the avaliable days and times for the given classroom in the Day and Time dropdown. Select a day and time for the class. Once you have selected a classroom and a day and time, you will then click the save button to lock this selection in place. Once the save button is successfuly pressed, you cannot edit that given selection. If you wish to change the selection, click on the edit icon that has appeared next to the save button. Once pressed, the selection that you have previously made will no longer be saved until you click the save icon again. Once you have selected and saved all conflict solutions, click the submit button at the bottom of the conflict page. This will save and send all the conflict solutions that you have completed. Once the conflicts have been submitted, you must generate the schedule again with the new changes that you have completed. If you decide to exit the conflicts page before you have submitted all your conflicts, only the conflicts that you have saved will be preserved, so make sure you save all the conflicts you wish to keep before closing out of the conflict resolution window.

How do you download the generated schedule?

After the schedule has been generated and all the conflicts that have arisen have been resolved, you can then download the schedule in CSV format. To do this, click on the download icon and the file will be immediately downloaded to your downloads folder.
